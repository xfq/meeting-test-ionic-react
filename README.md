# Demo Conference Application

This is a demo meeting app based on [Ionic React Conference Application](https://github.com/ionic-team/ionic-react-conference-app).

## Getting Started

* [Download the installer](https://nodejs.org/) for Node LTS.
* Install the ionic CLI globally: `npm install -g ionic`
* Clone this repository: `git clone https://github.com/ionic-team/ionic-react-conference-app.git`.
* Run `npm install` from the project root.
* Run `ionic serve` in a terminal from the project root.
* Profit. :tada:
